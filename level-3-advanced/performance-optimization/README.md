# Performance Optimization

Ce projet montre comment optimiser les performances d'une page web statique (landing page) en appliquant des techniques telles que la minification, l'optimisation des images, le lazy loading et le chargement différé des scripts.

## Structure du projet

- `index.html` : version originale non optimisée
- `style.css` / `script.js` : sources originales
- `style.min.css` / `script.min.js` : versions minifiées
- `index-optimized.html` : version intégrant toutes les optimisations
- `images/` : contient les icônes (PNG et WebP)
- `LIGHTHOUSE.md` : résultats des tests Lighthouse
